Electric Oven

Group Studio Project - Ankara, 2017


We modified TEKA’s existing in-built electric oven from a user-centered perspective. Our aim was to portray the oven as the heart of the kitchen and build interactions of the user with the oven at a more personal level.

^Intitial Brainstroming

^We observed a person at her home baking to get insight into the acitivity of using a product like an oven, and digging deeper into the rituals or peripheral expereinces that support it.

^These were some intital ideas for interfaces, each exploring a different type of interaction and look. With each user-test, we were observing the most used functions, time spent on each interaction, mood level, confusion level, tactility and eye movement.

^Final Design


The design incorporates portable tablet displays that can be detached and taken with the user so they don’t have to visit the kitchen repeatedly to monitor their food. The tablets can be customized from the oven touch screen and can display recipes, music, time, alarms etc. It allows the user to gift a recipe tablet to a friend who can then use it as a fridge magnet if he/she does not have the same oven. Pressing a tablet will automatically set the oven’s settings to the ones saved in the tablet. These are the primary interactions. It also can be further developed to have multiple intuitive exchanges between the user and the oven.
